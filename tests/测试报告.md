## 三、检测详情

### 3.1 SQL 注入 (CRITICAL)
- **检测方法**: skill_sqli_basic → skill_sqli_union
- **证据**:
  - 单引号注入触发 MySQL 错误
  - UNION SELECT 成功提取用户数据
- **攻击链**:
  1. nmap_scan 发现 3306 端口
  2. whatweb_scan 识别 MySQL 技术栈
  3. skill_sqli_basic 检测注入点
  4. skill_sqli_union 提取凭据

### 3.2 盲注 (HIGH)
- **检测方法**: skill_sqli_time_blind
- **证据**:
  - SLEEP(3) 延迟成功
  - 布尔盲注响应差异

### 3.3 反射型 XSS (MEDIUM)
- **检测方法**: skill_xss_reflected
- **证据**:
  - Payload 直接反射
  - 无输入过滤

### 3.4 存储型 XSS (HIGH)
- **检测方法**: skill_xss_stored
- **证据**:
  - Payload 存储于数据库
  - 访问时自动执行

### 3.5 命令注入 (CRITICAL)
- **检测方法**: skill_rce_command_injection
- **证据**:
  - `; id` 执行成功
  - 输出 `uid=33(www-data)`

### 3.6 文件包含 (HIGH)
- **检测方法**: skill_lfi_basic
- **证据**:
  - `../../../../etc/passwd` 成功读取
  - PHP 伪协议可用

### 3.7 文件上传 (CRITICAL)
- **检测方法**: skill_file_upload_testing
- **证据**:
  - 无文件类型限制
  - .php 上传成功

### 3.8 CSRF (MEDIUM)
- **检测方法**: skill_csrf_testing
- **证据**:
  - 密码修改表单无 Token
  - 跨站请求可执行
