  二、主循环工作流

  2.1 PentestAgent.execute() 执行流程

  1️⃣ 初始化
     ├─ ClaudeAgentOptions(system_prompt, model, working_dir)
     └─ ClaudeSDKClient.connect()

  2️⃣ 任务提交
     └─ await client.query(task)

  3️⃣ ⭐ 消息处理循环 (CORE LOOP)
     async for message in client.receive_response():
     ├─ TextBlock → output收集 + 旗标检测 + 步骤记录
     ├─ ToolUseBlock → 追踪工具开始 (tracer + event)
     └─ ResultMessage → 成本统计

  4️⃣ 旗标检测 (重要!)
     ├─ 6个正则模式: flag{}, FLAG{}, HTB{}, CTF{}, generic{}, 32-char hex
     ├─ 每次TextBlock都执行检测
     └─ 发出emit_flag()事件 → TUI更新

  5️⃣ 返回结果
     ├─ success: bool
     ├─ output: str (完整输出)
     ├─ flags_found: list[dict]
     ├─ cost_usd: float
     └─ walkthrough: list[str]

  2.2 AgentController的5态生命周期

  IDLE → RUNNING → [PAUSED ←→ RUNNING] → COMPLETED/ERROR

  状态转移:
  - pause() @ RUNNING → _pause_requested = True (在消息边界触发)
  - resume() @ PAUSED → _resume_event.set() (恢复等待)
  - inject() @ ANY → _pending_instruction = text (注入指令)

  关键设计: 消息边界是SAFE POINT (安全暂停点)
  - 在receive_messages()循环中检查_pause_requested
  - 不会在工具执行中断
  - 用户可输入指令后继续

  2.3 消息处理管道

  async def _process_message(msg, output_parts, flags_found):
      if msg.type == MessageType.TEXT:
          # 文本处理
          output_parts.append(msg.content)
          detected = _detect_flags(msg.content)
          for flag in detected:
              flags_found.append(flag)
              sessions.add_flag(flag, context)
              events.emit_flag(flag)  # ← TUI实时显示

      elif msg.type == MessageType.TOOL_START:
          events.emit_tool(status="start", name, args)

      elif msg.type == MessageType.TOOL_RESULT:
          events.emit_tool(status="complete", name, result)

      elif msg.type == MessageType.RESULT:
          sessions.add_cost(cost_usd)

  十、关键代码片段

  旗标检测 (核心)

  FLAG_PATTERNS = [
      r"flag\{[^\}]+\}",           # flag{...}
      r"FLAG\{[^\}]+\}",           # FLAG{...}
      r"HTB\{[^\}]+\}",            # HTB{...}
      r"CTF\{[^\}]+\}",            # CTF{...}
      r"[A-Za-z0-9_]+\{[^\}]+\}",  # generic{...}
      r"\b[a-f0-9]{32}\b"          # 32-char hex
  ]

  def _detect_flags(self, text: str) -> list[str]:
      flags = []
      for pattern in self.FLAG_PATTERNS:
          matches = re.finditer(pattern, text, re.IGNORECASE)
          for match in matches:
              flag = match.group(0)
              if flag not in flags:
                  flags.append(flag)
      return flags

  暂停/恢复 (简洁优雅)

  if self._pause_requested:
      self._set_state(AgentState.PAUSED, "Paused - waiting")
      await self._resume_event.wait()  # 阻塞
      self._resume_event.clear()

      if self._pending_instruction:
          await self.backend.query(self._pending_instruction)
